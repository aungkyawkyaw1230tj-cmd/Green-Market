User Policy & Rules

၁။ အထွေထွေ စည်းမျဉ်းများ: ဤ Platform ပေါ်တွင် တရားမဝင်သော ပစ္စည်းများ၊ လိမ်လည်မှုများနှင့် Spam များကို လုံးဝ ခွင့်မပြုပါ။

၂။ Escrow ဝန်ဆောင်မှု: ငွေပေးချေမှုများကို လုံခြုံစေရန် Admin မှ ကြားခံဆောင်ရွက်ပေးပါသည်။

၃။ Admin လုပ်ပိုင်ခွင့်: စည်းမျဉ်းချိုးဖောက်သော Post များကို Admin မှ အချိန်မရွေး ဖျက်ပစ်နိုင်ခွင့်ရှိပါသည်။

၄။ Report စနစ်: မသင်္ကာဖွယ်ရာ အကောင့်များ သို့မဟုတ် Post များကို တွေ့ရှိပါက Report ပြုလုပ်နိုင်ပြီး Admin ထံ တိုက်ရိုက်ရောက်ရှိမည်ဖြစ်သည်။
